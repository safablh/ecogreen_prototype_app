# EcoGreen ♻️

Mobile application prototype for waste recycling, built for a client project as demo.

🔗 **[Live Demo](https://safablh.github.io/ecogreen_prototype_app/)**

## About

EcoGreen encourages citizens to sort their waste (plastic bottles, paper) by rewarding them with points for every validated deposit. Points can be exchanged for real community rewards such as free swimming pool access, transport tickets, or parking discounts.

The app supports two user roles:

- **Citizens** — declare waste deposits and track their points.
- **Agents** — validate or reject deposits.

## Stack

- React
- Tailwind CSS

> This is a frontend only prototype: user accounts, points, and deposits are stored in the browser's `localStorage` (no backend/database).
